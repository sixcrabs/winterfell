# Starter

- app-spring-boot-starter : 适配 `springboot v2.x` 启动器, 封庄了常用基础功能
- app-spring-cloud-starter: 适配 `spring cloud` 启动器，在上述starter基础上封装cloud微服务注册发现等功能
- app-tongweb-starter: 适配 东方通环境
- app-javalin-starter: 适配 `javalin 4.x`框架启动器，封装了 `Ioc` 等功能
- app-vertx-starter: 适配 `vertx` 启动器，封装了依赖注入、配置解析等功能
- keystore-spring-boot-starter: 适配 `springboot v2.x` 启动器，封装了 `keystore` 功能
- srpc-spring-boot-starter: 适配 `springboot v2.x` 启动器，封装了 `srpc` 功能
