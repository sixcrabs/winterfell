# common backend

java 通用后端组件库

## 环境

- java 8
- springboot 2.7.18

> 部分组件在 springboot 3.5.x 下测试可用(如 `hutool-mini`,`keystore`)，但不保证全部可用

## 结构

- miscellaneous: 杂项，包含 `oss-client` 、`rpc`、`rest client`、 `keystore` 等
- samples： 示例工程代码
- starter: 适配`springboot v2` 等框架的快速集成starter
